# Random_password_generator
🔐 Random Password Generator (Python)

A simple and interactive Random Password Generator built using Python. This program allows users to generate secure passwords based on selected character types such as letters, numbers, and symbols.

---

📌 Features

- User-defined password length
- Option to include:
  - Letters (A–Z, a–z)
  - Numbers (0–9)
  - Symbols (!, @, #, etc.)
- Input validation for length and choices
- Generates strong and random passwords
- Displays password strength:
  - Weak
  - Medium
  - Strong

---

🧠 How It Works

- User selects password length
- Chooses character types
- Program combines selected characters
- Generates password randomly using Python libraries

---

▶️ How to Run

1. Make sure Python is installed
2. Save the file as "password_generator.py"
3. Run the program:

python password_generator.py
